ROI Equipment Comparison System

Overview

This project is a Java-based return on investment (ROI) comparison system designed to demonstrate operational planning logic, workflow automation, data structures, sorting algorithms, and user-driven equipment analysis.

The application allows users to build equipment comparison lists, calculate ROI values, sort investment options, and evaluate operational purchasing decisions using structured backend logic.

---

Features

- Equipment ROI calculations
- Dynamic comparison list creation
- User-driven workflow interaction
- Equipment cost and gain analysis
- ROI sorting and comparison logic
- Exception handling for invalid input
- Operational decision support workflows
- Custom equipment entry functionality

---

Technologies Used

- Java
- ArrayList
- LinkedList
- Collections & Sorting
- Comparable & Comparator
- Exception Handling
- Scanner Input Handling
- Conditional Logic
